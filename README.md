# godot_2d_rough_template
this is a godot project template for 2D game making.

this project contains the following: 

*basic player movement
<br>*camera script 
<br>*basic script for scene loading 
<br>*simple dialogue script
<br>*example scene node with Line2D (for collision) and static body

you can use this project for a foundation, adding your necessaries as you build up.
all assets provided in this project are free to use.
